# Practice-Make-Perfect

> Репозиторий с практическими работами (Python, Java, ИИ)

## Описание

Этот репозиторий содержит практические задания, выполненные мной, по следующим направлениям:

- **Python** — разные упражнения и проекты  
- **Java** — практики по Java (ООП, коллекции, паттерны, GUI)  
- **ИИ / машинное обучение** — задачи, реализованные через Jupyter ноутбуки  

Цель — собрать все в одном месте для удобного изучения, демонстрации и повторного использования.

## Как запустить / использовать

1. **Клонировать репозиторий:**  
   ```bash
   git clone https://github.com/MinAleDm/Practice-Make-Perfect.git
   ```
2. **Перейти в нужную папку:**

Для `Java`: `cd Practice-Make-Perfect/Java-main`

Для `Python`: `cd Practice-Make-Perfect/Python_pr`

Для `ИИ`: `cd Practice-Make-Perfect/Artificial-Intelligence\ Practice`

3. **Запустить проект / ноутбук:**

`.ipynb` файлы — через `Jupyter / VSCode`

`.java` файлы — через `IntelliJ IDEA` или другую `IDE`

`.py` файлы — через `Python`
