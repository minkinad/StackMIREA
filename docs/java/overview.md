# Java Практики

<img src="/java-icon.svg" alt="Java icon" width="56" />

Java-трек состоит из 24 практических задач и теперь ссылается на реальные реализации из `pr_Java/`, а не на шаблонные заглушки.

## Основные темы

- классы, объекты, инкапсуляция, наследование;
- интерфейсы, коллекции, обобщения и очереди;
- исключения и файловая система;
- GUI и паттерны проектирования;
- итоговый проект на основе системы заказов.

## Формат модулей

Каждый модуль содержит:

- теоретический блок;
- фрагмент кода прямо из решения;
- список файлов, участвующих в задаче;
- практический разбор;
- итоговый вывод.

## Исходники

Все задачи расположены в `pr_Java/`.
