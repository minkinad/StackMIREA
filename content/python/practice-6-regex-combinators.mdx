## Теория

Этот модуль больше про функциональные техники в Python, чем про классический regex:

- функции высшего порядка (`deriv`, `io`);
- функциональная имитация структур (`person`, односвязный список);
- комбинаторный подход к построению данных.

## Код из решения

```python title="pract6.ipynb (deriv + decorators)"
def deriv(f, h=0.0001):
    def fprime(x):
        return (f(x + h) - f(x)) / h
    return fprime

@io(input, input, input, print)
def f1(x, y, z):
    return x + y + z
```

```python title="pract6.ipynb (functional linked list)"
def pair(value, next_node=None):
    return lambda selector: value if selector == "value" else next_node

def head(lst):
    return lst("value")

def tail(lst):
    return lst("tail")

list_node = pair(1, pair(2, pair(3, pair(4))))
print("Head node value:", head(list_node))
```
