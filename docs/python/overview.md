# Python Практики

<img src="/python-icon.svg" alt="Python icon" width="56" />

Трек объединяет две линии материалов:

- классические практики по ООП, тестированию, ФВП и regex-комбинаторам;
- рабочие тетради по дисциплине ИИ (`Notebook1..8`) в формате MDX-документации.

## Что изучается

- объектно-ориентированное программирование;
- тестирование и контрактное программирование;
- функциональный стиль и комбинаторы;
- регулярные выражения и шаблоны обработки текста.
- NumPy / pandas / scikit-learn;
- базовые модели ИИ: регрессия, деревья, нейросети, кластеризация.

## Формат модулей

Каждая практика содержит:

- теоретический минимум;
- фрагменты реального кода из исходников;
- краткий технический разбор и контекст;
- вывод с фокусом на применении в следующих задачах.
