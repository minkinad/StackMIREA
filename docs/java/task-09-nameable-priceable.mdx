#### `Task9/Car.java`

```java title="Task9/Car.java"
package Task9;

public class Car implements Nameable{
    String name;
    public Car(String name){
        this.name = name;
    }
    @Override
    public String getName() {
        return name;
    }
    public String toString(){
        return "Car = "+ name;
    }
}

```
