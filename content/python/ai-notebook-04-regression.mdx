## Теория

Тетрадь фокусируется на аппроксимации данных и построении регрессионных моделей.

## Код из решения

```python title="Notebook4.ipynb (линейная аппроксимация)"
import numpy as np

x = np.array([0, 1, 2, 3])
y = [-1, 0.2, 0.9, 2.1]
A = np.vstack([x, np.ones(len(x))]).T
m, c = np.linalg.lstsq(A, y, rcond=None)[0]
```

```python title="Notebook4.ipynb (Simple Linear Regression)"
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

x_train, x_test, y_train, y_test = train_test_split(x, y, test_size=0.2, random_state=0)
regressor = LinearRegression()
regressor.fit(x_train, y_train)
y_pred = regressor.predict(x_test)
```

```python title="Notebook4.ipynb (multiple regression)"
from sklearn import metrics

regressor.fit(x_train, y_train)
y_pred = regressor.predict(x_test)
print('Mean Squared Error:', metrics.mean_squared_error(y_test, y_pred))
```
