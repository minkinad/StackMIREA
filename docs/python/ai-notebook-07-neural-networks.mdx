## Код из решения

```python title="Notebook7.ipynb (персептрон)"
class Perceptron:
    def __init__(self, N):
        self.w = [0 for _ in range(N)]

    def calc(self, x):
        return sum(self.w[i] * x[i] for i in range(len(self.w)))

    def sign(self, x):
        return 1 if self.calc(x) > 0 else -1

    def learn(self, la, x, y):
        if y * self.calc(x) <= 0:
            for i in range(len(self.w)):
                self.w[i] = self.w[i] + la * y * x[i]
```
